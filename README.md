## 📦 安装说明

### 方法一：自动安装（推荐）
1. 解压下载的压缩包
2. 双击运行 `install.bat`
3. 按提示完成安装

### 方法二：手动安装
1. 解压压缩包到任意目录
2. 直接运行 `{APP_EXE_NAME}.exe`

## 🚀 启动程序

安装完成后，您可以通过以下方式启动：
- 双击桌面快捷方式 "{APP_NAME}"
- 或直接运行程序目录中的 `{APP_EXE_NAME}.exe`

## 📋 系统要求

- Windows 7 及以上版本
- 至少 100MB 可用磁盘空间
- 网络连接（用于访问共享文件和检查更新）

## 🔧 功能特点

- 📦 零件库存管理
- 📥 入库记录管理  
- 📤 出库记录管理
- 👥 用户权限管理
- 📋 操作历史记录
- 🔄 在线自动更新

## ❓ 常见问题

**Q: 程序启动慢怎么办？**
A: 本版本已优化启动速度，如仍有问题请检查防病毒软件是否误拦截。

**Q: 无法访问数据文件？**
A: 请确保网络连接正常，并有权限访问共享目录。

**Q: 如何更新程序？**
A: 程序支持自动检查更新，也可以手动下载新版本覆盖安装。
